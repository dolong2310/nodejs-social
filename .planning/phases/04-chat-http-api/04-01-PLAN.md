## Objective

Triển khai **04-CONTEXT D-10, D-11**: điều chỉnh **read paths** social DB để khi có **block hai chiều**, nội dung post mà viewer **đã từng engage** vẫn đọc được nhưng **author = unknown user**; **không** mở rộng Phase 4 chat HTTP trong plan này.

## Scope

- In: repository helpers (engagement existence per `viewerId` + `postId`), mapper/redactor cho `PostDetailResponseDTO` / list DTOs / comment author / search hits; `PostsService`, `SearchService`; optional `UsersService` + `users` route cho target user bị block; cập nhật **REQUIREMENTS.md** (mục BLCK-02).
- Out: Chat DB, `/api/conversations` mới, Socket, notification.

## Non-goals

- Đổi rule **403** engagement (unverified) Phase 3.
- Reaction / edit message chat.
